# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Version 2.1.0] - 2025-11-28

### Added

- Added a sheet property editors.
- Added `grid-vertical` support.

### Changed

- The shortcut for _Show Spright editor panel_ was changed to <kbd>Ctrl-Shift-E</kbd>.
- The shortcut for _Complete Spright configuration_ was changed to <kbd>Ctrl-Shift-C</kbd>.
- Updated default spright version to 3.8.0.

## [Version 2.0.0] - 2025-11-18

### Added

- Added input and sprite property editors.

## [Version 1.3.0] - 2025-11-04

### Changed

- Updated default spright version to 3.7.0.

### Added

- Added filter input.
- Added checkbox for hiding image title.
- Showing tooltips for inputs and sprites.

### Fixed

- Improved performance for big configurations.
- Saving/restoring toolbar state.

## [Version 1.2.4] - 2025-01-04

### Changed

- Updated default spright version to 3.5.4.
